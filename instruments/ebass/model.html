Physical model

\begin{equation} \label{eq:model} \mu \frac{\partial^2 u}{\partial t^2} = EI \Delta^2 u+ T \Delta\left(u + 2\eta\frac{\partial u}{\partial t} \right) - 2\epsilon \frac{\partial u}{\partial t} \end{equation} String length is \(L\).

Losses

For \(EI, \eta, \epsilon \ll 1\), and \(u(t, x)=e^{-j \omega t} \sin\left(\frac{n\pi x}{L} \right)\) we get the equation \[-\mu \omega^2 = -T \left(\frac{n \pi}{L}\right)^2\left(1-j\eta\omega\right) +j \epsilon \omega\] so that \[\omega_n \approx\frac{n\pi}{L} \sqrt{\frac{T}{\mu}} + j \left(\frac{\eta T}{\mu}\left(\frac{n\pi}{L} \right)^2 + \frac{\epsilon}{\mu}\right)\] The \(\tau_{60}\) is therefore given by \[\tau_{60}(\omega_n) = \mu\frac{\ln(10^3)}{\eta \Re(\omega_n)^2 + \epsilon}\]
